# Welcome to Telegram UI

Dive into our sleek design on [Figma](https://figma.com/community/file/1348989725141777736/) to get a glimpse of what awaits you.

## Easy Installation

Getting started is a breeze with pnpm, npm or yarn. Simply run:

**pnpm:**

```sh
pnpm add telegram-ui-vue
```

**npm:**

```sh
npm install telegram-ui-vue
```

**yarn:**

```sh
yarn add telegram-ui-vue
```

## Smooth Development Process

Follow these simple steps to kickstart your development journey:

### 1. Import Styles

Before diving into the development, ensure to import the necessary styles:

```js
import 'telegram-ui-vue/dist/index.css';
```

### 2. Wrap Your App

Next, wrap your application with `AppRoot` to leverage our platform's features:

```jsx
<template>
  <AppRoot> // Your app </AppRoot>
</template>

<script setup lang="ts">
import 'telegram-ui-vue/dist/index.css';
import { AppRoot } from 'telegram-ui-vue';
</script>
```

### Usage example

```jsx
<template>
  <AppRoot>
    <Placeholder>
      <template #header>Title</template>
      <template #description>Description</template>
      <img
        alt="Telegram sticker"
        src="https://storage.mbredikhin.com/tg-banana.gif"
        style="display: block; width: 144px; height: 144px"
      />
    </Placeholder>
  </AppRoot>
</template>

<script setup lang="ts">
import 'telegram-ui-vue/dist/index.css';
import { AppRoot, Placeholder } from 'telegram-ui-vue';
</script>
```

With these steps, you're all set to explore the potential of our library. Happy coding!
