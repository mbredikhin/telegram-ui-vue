## Getting started

> [!TIP]
> Pnpm is used as a package manager in this project (`npm i -g pnpm`) and NodeJS >= 16 required

```bash
git clone git@github.com:mbredikhin/telegram-ui-vue.git
cd telegram-ui-vue
pnpm install
pnpm dev
```
